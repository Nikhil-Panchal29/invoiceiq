# 🚀 InvoiceIQ – AI-Powered Invoice Management Platform

InvoiceIQ is a full-stack AI-powered invoice management platform that automates invoice processing, validation, duplicate detection, analytics, reminders, and AI-assisted invoice queries.

---

## ✨ Features

- 📄 Invoice Upload & OCR Extraction
- 🤖 AI Assistant (Groq / Gemini)
- 🔍 Duplicate Invoice Detection
- ✅ AI Invoice Validation
- 📊 Dashboard & Analytics
- 📈 Spending Analysis
- 📧 Email Payment Reminders
- 🔐 JWT Authentication & Authorization
- 👤 User Authentication
- 📱 Fully Responsive UI
- ☁️ MongoDB Atlas Cloud Database

---

### Frontend

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Nikhil-Panchal29/invoiceiq.git
```

Move into the project

```bash
cd invoiceiq
```

---

## Backend

```bash
cd backend
npm install
npm run dev
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

## Backend (.env)

```env
PORT=5000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_jwt_secret

GROQ_API_KEY=your_groq_api_key

GEMINI_API_KEY=your_gemini_api_key

SMTP_HOST=smtp-relay.brevo.com

SMTP_PORT=2525

SMTP_USER=your_brevo_username

SMTP_PASS=your_brevo_smtp_key

SMTP_FROM=InvoiceIQ <your_email@example.com>
```

---

## Frontend (.env)

```env
VITE_API_URL=http://localhost:5000/api
```
